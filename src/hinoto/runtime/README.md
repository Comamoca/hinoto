# hinoto/runtime

A collection of code utilities for [Hinoto](https://github.com/Comamoca/hinoto),
designed for use with Node.js, Deno, and Bun.

</div>

<div align="center">

</div>

## 🚀 How to use

```gleam
// For Node.js
import hinoto/runtime/node

// For Deno
import hinoto/runtime/deno

// For Bun
import hinoto/runtime/bun
```

## ⛏️ Development

```sh
gleam format --check
gleam test
```
